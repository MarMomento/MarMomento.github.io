VSCode常用快捷键

1. 光标移动

作用 快捷键
多光标 Ctrl + Alt + Arrow
选中附近单词与重复内容 (Ctrl + D ) * number
选中多行内容并在末尾添加光标 Alt + Shift + i

2. 文件间移动

作用 快捷键
展示最近打开的文件列表 Ctrl + P
在新的编辑器中打开 Ctrl + P，Ctrl + Enter
跳转到指定行 Ctrl + G
跳转到指定功能 Ctrl + Shift + O
typescript 跳转到定义处 F12
跳转到函数实现处 Ctrl + F12

3. 代码

作用 快捷键
折叠/展开 Ctrl + Shift + [ / ]
全部折叠 Ctrl + K，Ctrl + 0

4. 搜索

作用 快捷键
文件内搜索内容,并跳转 Ctrl + F，Shift + Enter
大小写敏感 Alt + C
单词全匹配 Alt + W
正则 Alt + R
替换 Ctrl + H
全部替换 Ctrl + Alt + Enter
多文件搜索 Ctrl + Shift + F

5. 终端

作用 快捷键
显示/隐藏终端 Ctrl + ` / Ctrl + J
新建终端 Ctrl + Shift + `
拆分终端 Ctrl + Shift + 5
终端间的切换 Alt + Arrow

6. 编辑器

作用 快捷键
拆分编辑器 Ctrl + \
切换编辑器 Ctrl + number
调整文件所在的编辑器 Ctrl + Alt + Arrow

7. 工作区

作用 快捷键
放大/缩小 Ctrl + +/-
