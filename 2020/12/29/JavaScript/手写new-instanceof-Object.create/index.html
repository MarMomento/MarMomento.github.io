手写new-instanceof-Object.create

一、new

new运算符原理：

  1. 创建一个空对象，作为将要返回的实例对象
  2. 将该空对象的原型，指向构造函数的prototype
  3. 将该空对象赋值给函数内部的this
  4. 开始执行构造函数内部的代码
  5. 返回this：
    • 手动返回数值，会被忽略
    • 手动返回对象，则忽略this

二、instanceof

instanceof原理：依次查找自身原型链上的原型，看是否有匹配上的。

三、Object.create

Object.create：有两个参数，第一个参数为原型对象，第二参数为属性描述对象
