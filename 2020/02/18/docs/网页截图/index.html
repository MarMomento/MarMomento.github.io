一、网页截图

1. PC端

打开要截图的网页,F12(Fn+F12)打开控制台,选择Elements,ctrl+shift+p,输入shot即可选择截图方式

2. 手机端

打开要截图的网页,F12(Fn+F12)打开控制台

二、网页设计模式

在控制台输入document.designMode = 'on',即可修改页面(可以复制文字等…)
